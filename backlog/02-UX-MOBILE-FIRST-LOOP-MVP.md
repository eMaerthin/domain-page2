# Mobile-first UX Rules (MVP)

These are non-aesthetic constraints to maximize PV/user and reduce overhead.

## Quiz
- 1 question per screen.
- No multi-column layouts on mobile.
- After user answers:
  - immediately transition to next step (or result) with minimal delay.
- Insert ad slots:
  - between steps (every step or every N steps; configurable).
  - at result (always).

## Tools
- Primary screen:
  - input field(s)
  - single action button
  - output area
- Recommendations:
  - show next quiz/tools immediately after output.

## Ads
- Sticky bottom ads visible on mobile across quiz/tools flows.
- For desktop:
  - sidebar ad is always present.
